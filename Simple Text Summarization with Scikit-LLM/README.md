# Simple Text Summarization with Scikit-LLM

Nói là sử dụng Sckikit-LLM nhưng thực ra đang dùng huggingface do Scikit cần OpenAI mà cái đấy thì mất tiền
Về cơ bản cả 2 pipeline đều hoạt động như nhau chẳng qua kém chất lượng hơn.

Cũng không có gì đặc biệt lắm chỉ là call model về để chạy, giới hạn max và min length của summary.
