# Simple Local AI Agent with Ollama

Giống một repo mô phỏng cách AI Agent hoạt động hơn là 1 Agent thực sự
2 tool chính:
- Calculator tool: để tính toán các phép tính
- Knowledge base tool: để tìm kiếm thông tin trong knowledge base (kb lại là local chứ ko có db)

Chủ yếu dùng để học cách sử dụng LangChain để tạo ra 1 Agent đơn giản và cách hoạt động của AI Agent

link tham khảo: https://machinelearningmastery.com/building-ai-agents-with-local-small-language-models/
